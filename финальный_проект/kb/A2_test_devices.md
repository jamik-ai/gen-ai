# Category A: adding devices to a test environment

A request to add a mobile or server device to a test environment is handled within **2 business days**.
The request must include the terminal ID and the device's operating system.

The QA Lead approves the request. If the device needs to be connected to several machines at once
(cross-device test), the SLA extends to **3 business days**, since it requires sign-off from every machine
owner involved.
